Charm Editor User Manual
===================================

Welcome to Charm. Charm is a *structure-first* hex editor focused on aiding
reverse-engineering process of binary files. It allows you to add structure to
data interactively as you are able to recognize it, and alter how the data is
displayed to make it easier to read and understand once you have annotated the
structure.
